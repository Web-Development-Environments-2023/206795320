About Us

סבי דוד דהן בן אברהם דהן וסולטנה למשפחת בוחבוט נולד בשנת 1926 בעיר דימנאת שלמרגלות הרי האטלס דרומית לעיר מרקש.

סבתי מרים בת שרה (לבית סבאג) ובת עמרם לבית ברביבאי, נולדה גם היא בעיר דימנאת בשנת 1934 בעת ביקורה של אמה בעיירה לצורך חתונת אחיה. גדלה בעיר סטאט שבמרוקו. לימים חזרה עם אמה לעיר דימנאת לאחר מות אביה ושם הכירה את סבי והם התחתנו בעיר מרקש בשנת 1947.

הם בנו את ביתם בעיר דימנאת ונולדו להם שלושה ילדים: עמרם , אליהו ומרדכי. סבא התפרנס מתפירה והכנת נעליים וסבתא עזרה בפרנסה בתפירה וסריגה.

בשנת 1955 עלו לישראל עם שלושת ילדיהם מטעמי ציונות והותירו שם את משפחתם המורחבת. הם הגיעו למעברת משואה ליד בית גוברין ושוכנו בצריף פח. הקליטה היתה קשה מאוד ותנאי המחיה קשים מאוד.

לימים עברו למעברת גן יבנה ולאחר מכן לבתי קבע בגן יבנה. סבי התפרנס מעבודת תחזוקה בבסיס חצור הסמוך לגן יבנה וסבתי עזרה בעבודות מזדמנות כמו קטיף ונקיון.

בארץ נולדו להם עוד שישה ילדים: אברהם, שרה, מאיר, שמעון, שושי ועמי.

בשנת 1969 שכלו את בנם הבכור עמרם בתאונת אימונים של חטיבת גולני. למרות המשבר הקשה בחייהם השתקמו וגידלו את ילדיהם, דאגו לפרנסתם ולחינוכם מקטן ועד גדול.

כיום מתגוררים עדיין בגן יבנה מוקפים במשפחה אוהבת ותומכת וזכו לנכדים ולנינים רבים.

Q & A

שאלה: מנה שלוש הבדלים עיקריים המבדילים בין Web 1.0, Web 2.0, Web 3.0

תשובה:

Web 1.0 Web 2.0 Web 3.0
תוכן התוכן הוא לקריאה בלבד ומוצג באופן סטטי התוכן ניתן לקריאה ולכתיבה, והתוכן בו דינאמי גם לקריאה ולכתיבה, אבל בנוסף התוכן מותאם אישית למשתמש
מטרה המטרה העיקרית הייתה לספק אינפורמציה באופן מקוון התבסס על שיתוף ידע ואיפשור אינטראקציה חברתית ופעילות יומיומית הביא את עניין הקישוריות בין המידע שבאינטרנט, שיתוף ידע ומידע מקושר
חיפוש החיפוש מציג תוצאות מהירות, אבל לא תמיד מדוייקות, והתוכן שמוצג הוא מאקרו כולל מנועי חיפוש שמחזירים תגיות עם תוכן מיקרו החיפוש מדוייק יותר, מותאם אישית, ומשתמש באמצעיים טכנולוגיים כדוגמת בינה מלאכותית כדי לנתח את כוונת המשתמש בשאילתה


שאלה: הצג את ההבדלים בעזרת דוגמה

תשובה:

נציג דוגמה בה המשתמש כותב את השאילתא "1+1" במנוע החיפוש:
Web 1.0 ב-web1 יוצגו רק אתרים שבעליהם תייגו את השאילתא "1+1".
Web 2.0 ב-web2 כנראה ויוצג מחשבון עם השאילתא או שיוצג הפתרון, וכן אתרים אחרים שקשורים.
Web 3.0 ב-web3 כנראה שיוצא למשתמש גם את אתר תאגיד השידור "כאן" שמשתמש בסלוגן הזה לפרסום. כלומר השתמשו בבינה מלאכותית כדי להבין את כוונת המשתמש והציגו לא את התוכן שמותאם לו אישית.
